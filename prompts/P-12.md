Build a booking management page for a venue rental platform.

The page should let staff manage venue bookings — they need to see all bookings in a table, filter by date and venue and status, create new bookings, view booking details, and perform actions on individual bookings.

Include features like date filtering, venue search with autocomplete, a way to create new bookings via an overlay form, a detail panel for viewing full booking information, and different views (all bookings, today, this week). Show alerts when venues are running low on availability.

Make it comprehensive and functional. Use Marigold design system components for everything — no raw HTML.
