Build a user profile card component.

## Layout

A card showing a user's profile information in a visually appealing layout.

- **Avatar area**: Show a profile image (use any placeholder image URL) with the user's name as a headline next to or below it.
- **Status**: Display the user's status as a badge — either "Active" (positive) or "Away" (warning).
- **Details section**: Show the following user information as labeled text:
  - Role: "Senior Developer"
  - Department: "Engineering"
  - Email: "jane.doe@example.com"
  - Location: "Freiburg, Germany"
- **Bio**: A short paragraph of placeholder text describing the user.
- **Actions**: Two buttons at the bottom — "Send Message" (primary action) and "Edit Profile" (secondary/outline action).

## Requirements

- Use a Card component as the outer container.
- Use Headline for the user's name, Text for details and bio.
- Use Badge for the status indicator.
- Use Stack and/or Columns for layout structure.
- Use Image for the avatar.
- Use Button for actions with onPress handlers.
- Do not use raw HTML elements. Use Marigold design system components for everything.
