Build a settings page for an application.

## Layout

A single-page settings view organized into clearly separated sections using an accordion.

## Sections

### Profile Settings (accordion section 1)
- Display Name (required text field)
- Email Address (text field with type email)
- Bio (text area, optional)
- A "Save Profile" button

### Appearance (accordion section 2)
- Theme selection: a radio group with options "Light", "Dark", and "System Default"
- Language: a select dropdown with options "English", "German", "French"
- Compact Mode: a switch toggle with description "Reduce spacing and font size for denser layouts"

### Notifications (accordion section 3)
- Four checkboxes in a checkbox group:
  - "Email notifications for new messages"
  - "Push notifications for mentions"
  - "Weekly activity digest"
  - "Marketing and promotional emails"
- Notification Sound: a select dropdown with options "Default", "Chime", "None"
- A "Save Notification Preferences" button

### Privacy (accordion section 4)
- Profile Visibility: a radio group with options "Public", "Team Only", "Private"
- Show Online Status: a switch toggle
- Allow Search Engines: a switch toggle with description "Let search engines index your public profile"
- A "Save Privacy Settings" button

## Requirements

- Do not use raw HTML elements. Use Marigold design system components for everything.
- All interactive elements must be functional, not just visual.
