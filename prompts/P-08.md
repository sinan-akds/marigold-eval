Build a settings page for an application.

The page should let users configure their preferences across several areas — profile information, visual appearance, notification preferences, and privacy controls.

Group the settings into expandable sections so the page isn't overwhelming. Each section should have its own save action.

Include things like text inputs for profile info, toggles for on/off settings, radio buttons for exclusive choices, checkboxes for multi-select options, and dropdowns where appropriate.

Make it well-organized and usable. Use Marigold design system components, not raw HTML.
