Build a user profile card for a team directory.

A single card presents one person's profile:

- **Avatar** — a profile image (use any placeholder image URL) with the person's name as a heading beside or below it.
- **Status** — a status indicator showing either "Active" (positive) or "Away" (warning).
- **Details** — labeled lines of information:
  - Role: "Senior Developer"
  - Department: "Engineering"
  - Email: "jane.doe@example.com"
  - Location: "Freiburg, Germany"
- **Bio** — a short paragraph of placeholder text about the person.
- **Actions** — two buttons at the bottom: "Send Message" as the primary action and "Edit Profile" as a secondary, less prominent action.

## Requirements

- Build everything with Marigold design system components; do not use raw HTML elements (an image tag for the avatar is the only exception).
- Every interactive element must actually work, not just look the part.
